<h1>Pengaduan Siswa<h1/>
<br>
<h3>Deskripsi<h3/>
<p><b>Pengaduan Siswa<b/> adalah sebuah aplikasi web yang dirancan untuk memfasilitasi siswa dalam melaporkan pengaduan atau masalah yang mereka hadapi di lingkungan sekolah. Aplikasi ini memungkinkan siswa untuk dengan mudah menyampaikan keluhan secara online yang kemudian dapat direspon oleh pihak sekolah dengan lebih cepat dan terstruktur.
